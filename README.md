# 🎮 سكربت التسجيل الخارق لـ Scrcpy (ألعاب الأندرويد وببجي)
هذا الباتش يعتمد على برنامج Scrcpy المفتوح المصدر

مهم جدا: يتم التسجيل عن طريق كابل الـUSB الأصلي للجهاز ، فتح خيارات المطور ، تفعيل USB Debugging 
تم التجربة على جهاز redmi pad pro 5g كابل توصيل أصلي Type C 
باستخدام اداء ScrCpy اصدار 4.0
https://github.com/genymobile/scrcpy

هذا السكربت مصمم لتسجيل شاشة أجهزة الأندرويد والتابلت (مثل Redmi Pad Pro) بأعلى كفاءة ممكنة للجيمنج بدون تهنيج وبأعلى نقاء.

## ✨ المميزات:
- يطلب صلاحيات المسؤول (Admin) تلقائياً لضمان عدم سقوط الفريمات.
- يسجل بأبعاد الشاشة الأصلية وبسرعة 120 فريم وBitrate يصل لـ 64M.
- يقوم بحفظ الفيديوهات تلقائياً في مجلد `Output` وتسميتها بالتاريخ والثانية منعاً للمسح.
- يغلق نافذة العرض والتحكم لتوفير كامل طاقة الكمبيوتر والتابلت للتسجيل النظيف.

## 🛠️ طريقة الاستخدام:
1. قم بتحميل ملف `Run.bat` من هذا المشروع.
2. انقل الملف وضعه داخل المجلد الرئيسي لبرنامج **Scrcpy** الرسمي على كمبيوترك.
3. اضغط على ملف `Run.bat` كليكتين (Double Click).
4. ابدأ اللعب، وعند الانتهاء اغلق النافذة وستجد الفيديو في مجلد `Output`.
